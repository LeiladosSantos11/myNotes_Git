Acessing the Git Directory:
- ls -la or
- ls -l .git/

3 Project Stages:
- 1: Modified:
  -git init
  -git status
- 2: Staged:
  -git add.
- 3: Commited: 
  -git commited -m "add comment"

To Check the configuration on the file:
- git config -l

To track the commits:
- git log
